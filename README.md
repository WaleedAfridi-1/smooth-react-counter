# Smooth React Counter

A smooth and responsive counter application built with React using modern UI practices. This project demonstrates state management with React Hooks and a clean interface styled using Tailwind CSS. Ideal for beginners practicing React fundamentals.

---

## Features

- Increase counter value
- Decrease counter value
- React useState Hook
- Clean and modern UI
- Tailwind CSS styling
- Responsive layout

---

## Tech Stack

- React JS
- Tailwind CSS
- Framer Motion

---

## Installation

1. Clone the repository:
   git clone https://github.com/WaleedAfridi-1/smooth-react-counter.git

2. Go to project directory:
   cd smooth-react-counter

3. Install dependencies:
   npm install

4. Run the project:
   npm run dev
